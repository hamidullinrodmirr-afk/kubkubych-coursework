# КубКубыч — интернет-магазин LEGO-наборов

Курсовой проект по дисциплине «Веб-технологии».

**Автор:** Хамидуллин Радмир Робертович, группа 241-672.

## Что реализовано

- каталог наборов LEGO с поиском, сортировкой и фильтрацией по серии, цене, возрасту, наличию и скидке;
- JWT-аутентификация, личный кабинет и роли покупателя/администратора;
- корзина с контролем остатка и запретом дублирующих строк;
- оформление заказа: проверка адреса и суммы 500–100 000 ₽, атомарное уменьшение остатка, снимок состава и цен;
- жизненный цикл заказа `new → processing → shipped → delivered` и отмена;
- избранное, отзывы только после доставленного заказа, модерация отзывов;
- Django admin, Docker Compose, REST API.

## Запуск

```bash
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
python manage.py migrate
python manage.py seed_store
python manage.py runserver
```

Откройте `http://127.0.0.1:8000/`. Демо-администратор: `admin@kubkubych.ru` / `admin123`.

## Основные API

`/api/products/`, `/api/cart/items/`, `/api/orders/`, `/api/reviews/`, `/api/auth/`.

Администратор управляет наборами через `/api/products/` и статусом заказа через `PATCH /api/orders/{id}/status/`.
